Adding code from local repo to online repo


The way I managed to do it which is pretty easy is as follows:

  1. Navigate to directory in cmd/terminal.
  2. Once there type "git init"
  3. Edit your files etc
  4. Once done tpye in cmd/terminsl "git add *"
  5. Then type in "git commit -m "Message_you_want_for_Commit""
  6. Then type "git push origin master". This will push all your changes to your repo.
